# Experiment Template Pointer

This repository now uses folder-based experiment planning.

For each substantial attempt, create a new folder under `docs/experiments/` using this naming pattern:

- `YYYY-MM-DD_HH-MM_vNN_short-slug`

Example:

- `2026-03-29_20-01_v01_ssflv6-mask-overhead`

Then copy the template files from:

- `docs/experiments/_template/PLAN.md`
- `docs/experiments/_template/SPEC.md`
- `docs/experiments/_template/WORKLOG.md`
- `docs/experiments/_template/RESULT.md`

Read `docs/experiments/README.md` for the operating rules.
